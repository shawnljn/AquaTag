# Companion software

AquaTag includes research-facing and swimmer-facing tools:

- [`annotation-tool/`](annotation-tool/) synchronizes video playback with IMU plots for session review and event marking.
- [`mobile-app/`](mobile-app/) connects to AquaTag over BLE, supports device setup, and transfers recorded sessions.

Each application will include its own environment, build, test, and file-format documentation.
