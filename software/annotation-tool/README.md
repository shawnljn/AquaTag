# Annotation tool

The AquaTag annotation tool displays synchronized video beside accelerometer and gyroscope traces so researchers can inspect recordings frame by frame and mark swimming events efficiently.

Planned release contents:

- Application source and dependency lockfile
- Launch and packaging instructions
- Supported video and sensor-log formats
- Clock-alignment and synchronization workflow
- Event definitions for stroke type, swimming state, and turns
- Keyboard shortcut reference
- Import and export file specifications
- Review and adjudication workflow

Example recordings will not be bundled with the application. Users can load their own locally stored files.
