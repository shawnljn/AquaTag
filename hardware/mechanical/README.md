# Mechanical design

Planned contents:

- Editable CAD source for the enclosure and integrated clip
- STEP files for interchange and STL/3MF files for printing
- Head/goggle-strap, wrist, and lower-back mounting adapters
- Dimensioned drawings and required tolerances
- Print orientation, supports, layer height, wall count, and material profiles
- Photos/renders that identify each revision

The paper evaluates a sharp rectangular enclosure as the fabrication-efficient baseline and discusses rounded-rectangle and capsule-like variants. Clearly label which geometry was used in each validation result. Do not infer comfort, lifetime, or sensing performance for untested variants.
