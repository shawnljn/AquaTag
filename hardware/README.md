# Hardware

This directory will contain the reproducible AquaTag hardware release.

- [`electronics/`](electronics/) — schematic, PCB, BOM, placement, and manufacturing outputs
- [`mechanical/`](mechanical/) — enclosure, clip, mounting adapters, and printable files
- [`fabrication/`](fabrication/) — assembly, epoxy potting, charging-contact care, and pressure testing

Hardware releases should use matching version tags across these three areas and state the compatible firmware revision.
